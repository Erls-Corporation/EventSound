EventSound is an interface aimed at teaching how events work in the DOM and how they can be used in jQuery.

The main characteristics about events in Web pages is that some of them bubble in the DOM: they do not only occur on the target element but also on its parents and its parent's parent... all of its ancestors.

On the left panel, the scene, the bubbling phase has been decomposed to be visualizable.

The "bubble beat" can be modified with the following slider if it is too slow.
